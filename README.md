# Procédure à faire pour installer les fichiers sur un serveur local XAMPP:

- Installer XAMPP  
- Copier mon dossier (site web complet) **sae105** dans: *C:\\xampp\\htdoc\\*  
- Lancer le panneau de contrôle XAMPP.  
- Démarrer le serveur Apache  
- Autoriser l’accès aux port 80 et 443   
- On peut ensuite accéder à notre site avec l’adresse: *http://localhost/sae105*
